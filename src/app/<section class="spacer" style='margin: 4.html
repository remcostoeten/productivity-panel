Developer Guide

Here’s everything you need to connect your website to this repository.

Install the basehub SDK
Used to fetch data from BaseHub. It includes utils that help you display it.

Terminal 

npm install basehub
Configure
Set required environment variables.

.env 

1BASEHUB_TOKEN="bshb_pk_****"
2

3# optional
4

5BASEHUB_DRAFT=<true|false> # defaults to false
6BASEHUB_REF=<branch-name|commit-id> # defaults to your default branch
Generate
Use the basehub script to generate a type-safe SDK for your app.

Terminal 

npx basehub

Important: Make sure you run the generator before your app's build step. The recommended setup for Next.js looks like this:

package.json 

1{
2  "scripts": {
3    "dev": "basehub dev & next dev",
4    "build": "basehub && next build",
5  }
6}
Integrate
Import it into your app and retrieve your data without worrying about type definitions.

app/page.tsx 

1import { Pump } from 'basehub/react-pump'
2import { draftMode } from 'next/headers'
3

4const Page = async () => {
5  return (
6    <Pump
7      queries={[{ __typename: true }]}
8      next={{ revalidate: 30 }}
9      draft={draftMode().isEnabled}
10    >
11      {async ([data]) => {
12        'use server'
13        return <>{JSON.stringify(data, null, 2)}</>
14      }}
15    </Pump>
16  )
17}
18

19export default Page
